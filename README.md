# 💇‍♀️ Salon Management System

A web-based Salon Management System built using Django framework.

## 🚀 Features

- User Login & Authentication
- Admin Dashboard
- Appointment Booking
- Appointment List View
- Clean UI using HTML templates

## 🛠️ Technologies Used

- Python
- Django
- HTML
- SQLite3

## 📂 Project Structure

- salon/ (Main Project Folder)
- salonapp/ (Application Folder)
- templates/ (HTML Files)
- manage.py

## ⚙️ How to Run This Project

1. Clone the repository
2. Install dependencies
3. Run:
   python manage.py runserver
4. Open browser and go to:
   http://127.0.0.1:8000/
